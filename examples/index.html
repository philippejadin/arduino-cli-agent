Arduino-cli server

Here are your options :

  • Check the output of the arduino-cli command line to verify it's installed
  • Use the compiler sample form

API

  • GET /board/list List connected boards
  • GET /board/listall List available (installed) boards
  • GET /core/install/ Install a package named
  • GET /core/list> List available (installed) cores
  • GET /core/update-index> Update core index (from arduino repository)
